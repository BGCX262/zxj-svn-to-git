ZX taglib.

This is a convience library of JSP developers wanting to create custom JSP pages.

Here is some example JSP custom tags that could be used. One for initialising the zX object and another to do the logging. These JSP custom tags are more of a proof of concept than some thing worthwhile using.

However the idea is to extend these and allow for the creation of table tags and form tags etc..
